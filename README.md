
# R-TYPE

Main C++ project in 3rd year at Epitech.
The R-TYPE is a mutliplayer game based on existing R-Type game.

The project is split in 3 parts:

 - Client side
 - Server side (in UDP and TCP protocol)
 - Game Engine
